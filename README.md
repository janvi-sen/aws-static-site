## 💻 Technologies Used
- HTML5
- CSS3
- JavaScript
- AWS S3 (Static Website Hosting)

## 🚀 Deployment Steps

1. Create an AWS S3 bucket
2. Enable Static Website Hosting
3. Upload `index.html`, `style.css`, and other files
4. Make files public
5. Access your site using the S3 website endpoint
